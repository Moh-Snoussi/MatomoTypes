# MatomoType

## Description

A npm package that contains the types of Matomo (or piwik) tracking library.

## Getting Started
### Installing
```shell
npm install --save matomotypes
```

### Usage
```typescript
import {Matomo} from "matomotypes"

const _paq : Matomo.TrackingCollector = [];
```
